# 🌡️ Temperature & Humidity Sensor Project

This project is a **Temperature and Humidity Monitoring System** built using the **PIC16F877A microcontroller** and **DHT11 sensor**, with output displayed on an LCD. It is designed for educational and environmental monitoring purposes.

---

## 🔧 Components Used

| Component         | Description                            |
|------------------|----------------------------------------|
| PIC16F877A       | Microcontroller (40-pin)               |
| DHT11            | Digital temperature and humidity sensor |
| 16x2 LCD Display | For showing temperature and humidity    |
| Resistors, Capacitors | As per circuit design               |
| Power Supply     | 5V regulated power (via PCB)           |
| PCB              | Custom designed and fabricated         |

---

## 🛠️ Software Tools

- **MPLAB X IDE** – Code development and simulation
- **Proteus** – Circuit simulation
- **PIC C Compiler (MPLAB XC8)** – C coding for PIC
- **PCB Design Software** – For board design and layout

---

## ⚡ Circuit Power

The system is powered using a **regulated 5V supply** via a custom-designed **PCB**. The DHT11 sensor provides digital data to the microcontroller, which is processed and shown on the LCD.

---


## 💡 How It Works

1. DHT11 sends temperature & humidity data to PIC16F877A.
2. Microcontroller processes the data.
3. Output is displayed on a 16x2 LCD.
4. The entire system runs on a 5V regulated power supply.
